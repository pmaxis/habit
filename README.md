## How to start

Backend:

```bash
cd backend
docker compose up -d
npm install
npm run start:dev
```

Frontend:

```bash
cd front
npm install
npm run dev
```

Open [http://localhost:5174/]( http://localhost:5174/) in browser.
Swagger: [http://localhost:3001/api](http://localhost:3001/api)  in browser.
